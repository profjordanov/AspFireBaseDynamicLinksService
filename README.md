## A Web service that integrates Firebase Dynamic Links API.

More information for dynamic links can be found [HERE](https://firebase.google.com/docs/dynamic-links/) .

> Note: If you want to try it out right now, you'll have to registrate [HERE](https://firebase.google.com/) . Then add new project. You will nedd also to generate yourself some [Google Console](https://console.cloud.google.com)

generate yourself some [Google Console](https://console.cloud.google.com) credentials and build it yourself. Just download your `credentials.json` file and place it at the root of your project.
